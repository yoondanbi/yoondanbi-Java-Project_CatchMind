## 🚩 프로젝트 소개
**Java를 이용한 그림 맞추기 게임**
  - 사용자가 번갈아가며 그림을 그리고 맞히는 방식의 멀티플레이 게임이다.
  - 각 참가자는 자신만의 닉네임을 설정할 수 있고, 준비 버튼을 통해 게임 시작 여부를 조율한다.
  - 출제자는 주어진 제시어를 보고 그림을 그리고, 다른 참가자들은 그 그림을 보고 제시어를 추측해야 한다.
  - 라운드가 끝나면 역할이 자동으로 바뀌며, 모든 플레이어가 공평하게 출제자 역할을 수행하게 된다.
  - 게임은 점수제로 진행되며, 정답을 맞힐 경우 점수가 올라간다.
